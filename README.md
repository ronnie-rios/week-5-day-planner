# Work Day Scheduler 

## Description
A simple day planner where one can enter an event and it will save to local storage for a 9 to 5 work day.

## Technologies Used
* HTML
* JavaScript
* CSS
* Moment JS
* JQUERY 

## Installation
To install, visit the repository and download the files.
